# MonPremierProjet

Application Angular générée avec Angular CLI v21.0.5.

Prerequis
- Node.js et npm installés
- (Optionnel) Angular CLI : `npm install -g @angular/cli`

Installation
```bash
npm install
```

Lancer le serveur de développement
```bash
ng serve
```
Ouvrir http://localhost:4200/ — l'application se recharge automatiquement lors de modifications.

Construire pour la production
```bash
ng build
```
Les fichiers compilés sont placés dans le dossier `dist/`.

Tests unitaires
```bash
ng test
```
(Le projet utilise Vitest pour les tests unitaires.)

Tests end-to-end
```bash
ng e2e
```

Ressource
- Documentation Angular CLI : https://angular.dev/tools/cli
